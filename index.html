How to use this library is referance call as "xsl:include" for your XSLT.

実装状況ドキュメント　Implement Document

定義済みグローバル変数　Predefined global variables

  • $BIG_NUM = 10^(16)

    ゲタ　（計算精度を上げるための数）
    Big number(for increase precision number)

  • $MAX_VALUE = 10^(308)

    最大値　（計算には使えない）
    Max value on XSLT (don't use calculation number)

  • $Math_PI = 3.14159265358979

    円周率π
    Ludolph's constant.

  • $Math_E = 2.718281828459045

    ネイピア定数e
    Napier's constant.

  • $Math_LN2 = 0.693147180559945

    2の自然対数
    natural logarithm 2.

実装済み関数　Implemented formulas

  • (double) #text = cos[(double) arg]

    "arg"に対する余弦を返す
    return of the cosine value of "arg".

  • (double) #text = atan[(double) arg]

    "arg"に対する逆正接を返す
    return of the arc-tangent value of "arg".

  • (double) #text = exp[(double) arg]

    "arg"に対する自然指数を返す
    return of the natural exponent value of "arg".

  • (double) #text = cosh[(double) arg]

    "arg"に対する双曲線余弦を返す
    return of the hyperbolic-cosine value of "arg".
    "arg"が計算範囲内でなければ空文字となる
    if value of "arg" was out of range, than empty text.

  • (double) #text = sqrt[(double) arg] arg > 0

    "arg"に対する平方根を返す
    return of the square root value for "arg".
    "arg"が0以上でなければ空文字となる
    if value of "arg" was out of range, than empty text.

  • (double) #text = ln[(double) arg] arg > 0

    "arg"に対する自然対数を返す
    return of the natural logarithm value for "arg".
    "arg"が0以上でなければ空文字となる
    if value of "arg" was out of range, than empty text.

  • (double) #text = sin[(double) arg]

    "arg"に対する正弦を返す
    return of the sine value for "arg".

  • (double) #text = tan[(double) arg]

    "arg"に対する正接を返す
    return of the tangent value for "arg".

  • (double) #text = acos[(double) arg] ( -1 < arg < 1 )

    "arg"に対する逆余弦を返す
    return of the arc-cosine value of "arg".
    "arg"が-1以上1以下でなければ空文字となる
    if value of "arg" was out of range, than empty text.

  • (double) #text = asin[(double) arg] ( -1 < arg < 1 )

    "arg"に対する逆正弦を返す
    return of the arc-sine value of "arg".

  • (double) #text = tanh[(double) arg]

    "arg"に対する双曲線正接を返す
    return of the hyperbolic-tangent value of "arg".
    "arg"が計算範囲内でなければ空文字となる
    if value of "arg" was out of range, than empty text.

  • (double) #text = sinh[(double) arg]

    "arg"に対する双曲線余弦を返す
    return of the hyperbolic-sine value of "arg".
    "arg"が計算範囲内でなければ空文字となる
    if value of "arg" was out of range, than empty text.

  • (double) #text = log[(double) base , (double) arg] ( arg > 0 , base > 0 )

    "base"を底とする"arg"の対数を返す
    return of the logarithm whose base is "base" for "arg".
    "arg"または"base"が0以上でなければ空文字となる
    if value of arg or base was out of range, than empty text.

  • (double) #text = acosh[(double) arg] arg > 1

    "arg"に対する逆双曲線余弦を返す
    return of the area-hyperbolic-cosine value of "arg".
    "arg"が1以上かつ計算範囲内でなければ空文字となる
    if value of arg was out of range, than empty text.

  • (double) #text = asinh[(double) arg]

    "arg"に対する逆双曲線正弦を返す
    return of the area-hyperbolic-sine value of "arg".
    "arg"が計算範囲内でなければ空文字となる
    if value of "arg" was out of range, than empty text.

  • (double) #text = atanh[(double) arg] ( -1 < arg < 1 )

    "arg"に対する逆双曲線正接を返す
    return of the area-hyperboli-tangent value of "arg".
    "arg"が-1以上1未満かつ計算範囲内でなければ空文字となる
    if value of "arg" was out of range, than empty text.

  • (double) #text = pow[(double) base , (double) arg] ( base > 0 )

    "base"を底とする"arg"の指数を返す
    return of the exponent whose base is "base" for "arg".
    "base"が0以上でなければ空文字となる
    if value of base was out of range, than empty text.

  • (int) #text = sgn[(double) arg]

    "arg"が正の実数なら1、負の実数なら-1、いずれでもなければ0を返す
    return if value of "arg" is a positive real number than 1,
    else if value of it's a negative real number than -1,
    otherwise value of it's 0.

  • (double) #text = abs[(double) arg]

    "arg"に対する絶対値を返す
    return of absolute value for "arg".
